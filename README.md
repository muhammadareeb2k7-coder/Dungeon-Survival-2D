# 2D Dungeon Survival Game
A 2D Dungeon Survival Game developed in Unity featuring combat, NPC interactions, missions, loot boxes, and a final boss battle.

## Description
This project is a 2D Dungeon Survival Game developed using Unity. The game focuses on survival gameplay where the player explores a dungeon environment, fights enemies, completes missions, and defeats a final boss at the end of the level.

This project was created as part of my learning in game development while studying at the National University of Technology (NUTECH). It helped me improve my understanding of game mechanics, programming logic, and Unity development.

## Features
- Combat system with multiple enemies
- NPC interactions
- Score system to track player performance
- Loot boxes that provide rewards
- Mission-based gameplay
- Final boss battle at the end of the mission

## Technologies Used
- Unity Game Engine
- C# Programming Language

## Gameplay
In this game, the player navigates through a dungeon environment while fighting enemies and completing missions. Players can interact with NPCs, collect loot boxes for rewards, and increase their score. At the end of the mission, the player must defeat a boss to complete the level.
